# Лабораторна робота №3

## Тема 
Git та Linux CLI

## Модель
Математичне моделювання оптимального розкрою рулонної тканини для виробництва футболок (5 семестр)

## Автор
Жарук Данил, Холудієв Денис, група АІ-235

## Опис
Програма вирішує задачу оптимального розкрою рулонної тканини з урахуванням виробничого попиту.
Автоматично генерує шаблони без залишків та знаходить мінімальну кількість рулонів для розмірів XS, S, M, L, XL.
Код написаний на [Python].
