## Модель
Математичне моделювання оптимального розкрою рулонної тканини для виробництва футболок (5 семестр)

## Автор
Жарук Данил, Холудієв Денис, група АІ-235

## Опис
У роботі реалізовано математичну модель оптимального розкрою рулонної тканини для виробництва футболок різних розмірів.
Програма автоматично генерує всі можливі шаблони розкрою без залишків та дозволяє виключати недоступні шаблони через технічні обмеження.
На основі введеного виробничого попиту модель знаходить мінімальну кількість рулонів, що повністю задовольняє попит по кожному розміру (XS, S, M, L, XL).
Задача формулюється як цілочисельне лінійне програмування та розв'язується за допомогою бібліотеки PuLP.
Результати відображаються у вигляді стовпчикової діаграми для зручного аналізу розподілу шаблонів.
Код написаний на [Python].
